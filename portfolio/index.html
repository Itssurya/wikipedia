Biography

Surya A is a Python Backend Developer with 1.7+ years of experience building production-ready RESTful APIs using Flask and FastAPI. He focuses on designing scalable, secure, and well-structured backend services that power real-world applications.

His expertise spans authentication (JWT and token-based auth), role-based access control, SQL database design with SQLAlchemy, and backend performance optimization. He enjoys debugging complex production issues and improving reliability, logging, and observability for backend systems.

Surya is comfortable working across the stack when needed, from API design to database schema modeling and basic cloud deployments with Docker and AWS. He also experiments with Generative AI (LLMs and RAG) as a value-add layer on top of backend services.

Outside of day-to-day engineering, he builds side projects that combine clean backend architectures with AI-powered features, such as code review assistants and retrieval-augmented assistants.
